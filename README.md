## 📌 Project Overview  
This project analyzes real-world **NYC Taxi trip data** to extract meaningful insights about ride demand, pricing patterns, and customer behavior.  

The analysis is performed using **Python (EDA)** and visualized through an interactive **Power BI dashboard**, turning raw data into actionable business insights.

## 📈 Key Insights  

🚦 **Peak Hours:**  
- Ride demand increases throughout the day and peaks during **evening (5–7 PM)**  

📅 **Peak Days:**  
- Higher ride activity observed during **mid-week (Wed–Thu)**  

💸 **Fare Trends:**  
- Strong positive correlation between **trip distance and fare amount**  

📊 **Revenue Drivers:**  
- Trips between **5–10 km** contribute significantly to total revenue  

💡 **Customer Behavior:**  
- Higher tips are observed in **short to mid-distance trips (2–10 km)**  

📆 **Weekday vs Weekend:**  
- Slightly more rides on **weekdays (~51%)** compared to weekends
